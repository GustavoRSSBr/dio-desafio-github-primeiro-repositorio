Tipos de dados - valor, conjunto de valores que pode assumir e consequentemente as operações que podem ser realizada com o mesmo. (junção das 3 coisas)

Estática(forte) - a linguagem obriga você a definir o tipo da váriavel durante sua aplicação

Dinamica(fraco) - Não é obrigado a definir o tipo na criação.

Primitivo - tipos de dados mais básicos atônicos. Numeral ou textual 
Composto - composto de varios tipos de dados (primitivos ou compostos)

devemos declarar de acordo com a finalidade
tipos de dados: Todos começam com um valor default

textual - char(16 bits) usa aspas simples
String - textos e usa aspas duplas

numeral (byte = -128 a 127) short (-32768 a 32767) int ( -2 bilhoes a 2 bilhoes)
long (64 bits) float (32 bits e decimais) double (64 bits e decimais)
